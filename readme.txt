== Description ==

WTIpress makes a Wordpress site multilingual, and lets you translate posts and pages using the [WebTranslateIt.com](https://webtranslateit.com) service.

== Installation ==

1. Get an account at [WebTranslateIt.com](https://webtranslateit.com). It’s a powerful web-based translation tool.
2. Download the latest release here.
3. Unzip it, and place it in the wp-content/plugins folder of your blog.
4. Browse the plugins section of your blog and activate the plugin.
5. Create a project on WebTranslateIt.com, copy the API key from the project settings and paste on WTIpress’s setup page.
